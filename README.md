# 🛒 Simple E-commerce Store

A full-stack e-commerce web application built using React, Node.js, Express.js, and MongoDB.

## 🚀 Features

- User Registration & Login
- JWT Authentication
- Product Listings
- Product Details Page
- Shopping Cart
- Order Placement
- Admin Product Management
- Responsive Design

## 🛠️ Tech Stack

Frontend:
- React.js
- Tailwind CSS

Backend:
- Node.js
- Express.js

Database:
- MongoDB

Authentication:
- JWT
